TanStack Query\
TailwindCss\
React Hook Form\
Orval

Swagger
### Extension:
React\
Vite\
FSD\
TanStack Query\
TailwindCss\
Orval

## Installation

1. Clone the repo:

```bash
  git clone https://github.com/k1really/net-ninja.git
```

2. Install NPM packages:

```bash
  cd server(client, extension)
  npm install
```

3. Start projects:

```bash
  client:
  npm run dev

  server:
  npm run start:dev

  extension:
  npm run dev
```
